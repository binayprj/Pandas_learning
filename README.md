# Pandas_learning
This repository contains my practice projects and data analysis exercises.

## Project 1: Netflix Movie Duration Analysis
Investigating Netflix movie durations during the 1990s using Python and Pandas.
* **Key Skills:** Data filtering, looping through DataFrames, and Matplotlib visualization.
  
## Project 2: NYC SAT Performance Analysis
Investigating SAT scores across NYC public schools to identify top performers and analyze score variability by borough.

* **Key Skills:** Data aggregation with `.groupby()` and `.agg()`, calculating standard deviation, and sorting data for performance insights.
* **Key Insight:** Discovered which borough has the highest inequality in test scores using statistical spread.

## Project 3: Nobel Prize Winners Analysis
Analyzing trends in Nobel Prize awards, including gender distribution and the most common birth countries.
* **Key Skills:** Data cleaning, value counts, and temporal analysis with Pandas.
